OpenGL assignment2 program with the following keyboard functionality:
Key d: pause
key r: start a game or resume the game if it is paused
key right/left arrow: move the space ship right or left
key space: fire a missile.
key q and escape: quit the program.

This game has two levels, if pass the first level, it will go to level two.
In the level two, there are more alians and bombs.
